Coding Experience

Introduction

C# was my first exposure to the world of coding. I took an intro to C# class at a small college called Northwestern Michigan College in Traverse City, Michigan. There my professor John Velis really sold coding and made it fun, fun enough that I would later make Computer science my major. Pursuing this degree has encouraged me to learn many other different languages to broaden my skills in the computer science field, some languages ive learned are obscure and not well known and others are extremely well known and documented. In this section im going to go more in depth into some of the more well known languages ive gotten to know and how ive used them in classes.

C#

My first ever class was in the language of C#. While learning C# I made larger and larger projects as required from the class with some free range to pick what projects we wanted to work on. I opted to make a small ATM for my first project. This ATM had a way to make an account and password and would require you to login in order to withdraw or deposit any hypothetical money.


Later on when I took the more experienced C# class or projects got larger and more complex. During that semester we got the opportunity to code with "Finch" robots. These robots had many different sensors on them to monitor the world around them. Along with those they also had the ability to make noise and move around form two small wheels. Our final project for the second C# class was to make a text based quest game. My game was based around the Star Wars franchise where it prompted the player to enter their name, age, what race from star wars they would like to be called, and what side of the force they would like to choose. The initial game would load where the player could move from planet to planet, interact with NPCs fight bad NPCs, and collect parts for a lightsaber along with some other items to aid them along their journey.


Java

When I transferred colleges to Grand Valley State University the first classes I had to take were intro classes for the Java language. Because C# is very similar to Java, this transition was pretty simple. In the more experienced Java class we were tasked to make a Tic Tac Toe game that would prompt the player to enter what size of a board they wanted from 3x3 up to 15x15, and how many in a row they wanted to be a win. Once we got that working we were tasked to create an opponent that would attempt to block you from getting a win. Overall a very fun project and the end product was semi hard to beat the CPU which made it all worth while.

I got a ton of real world experience with SQL and it allowed me to grow my knowledge of the language greatly.
